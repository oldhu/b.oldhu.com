It's Windows XP Embedded

Nov 22, 2005
Not XP, but XP Embedded version 2002. And has the following characteristics. 1. Cannot see all the memory capacity. memory usage is restricted by boot.ini parameter /MAXMEM=407 2. Almost all modules are "Non-p&p drivers" 3. A lot of unknown devices are listed in the device manager tree. 4. Cannot finger out how system drives are mirrored. 5. There is another boot.ini parameter says /hal=hal-i0.dll, which means the HAL is replaced. 6. pagefile size is 784M 7. system drive size is 2.75GB
