Adobe Reader could not install correctly on Mac OS X

Mar 21, 2009
First of all, I just noticed Adobe has changed the name of Acrobat Reader to Adobe Reader...

Today when I run Adobe Reader for the first time after I installed it on Mac OS X,  it says:

"Adobe Acrobat could not install correctly. Adobe Acrobat was unable to install and must quit."

I googled around and found the following link very helpful:

http://kb.adobe.com/selfservice/viewContent.do?externalId=321485

I recall days before I deleted all printers' drivers from my Mac... and the whole /Library/Printers folder. After recreate an empty /Library/Printers folder, Adobe reader goes normal.
